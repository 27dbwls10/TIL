# 그라데이션

1. xml 파일 drawable에 생성
2. 가장 상단 코드 수정
    ```kotlin
    <shape xmlns:android="http://schemas.android.com/apk/res/android"
        android:shape="rectangle" >
    ```
3. `<shape>` 안에 `<gradient />` 추가하고 안에 내용은 아래 종류에 따라 추가


## Linear

- 일직선으로 색이 퍼짐
- 9시부터 반시계방향으로 0~360

```kotlin
<gradient
    android:angle="90"
    android:endColor="@color/main_01"
    android:centerColor="@color/main_02"
    android:startColor="@color/main_03"
    android:type="linear" />
```

## Radial

- 원형으로 색이 퍼짐

```kotlin
<gradient
    android:endColor="@color/main_01"
    android:centerColor="@color/main_02"
    android:startColor="@color/main_03"
    android:type="radial" />
```

## Sweep

- 링에 색을 입힐때 자주 사용
- 점 주면으로 색이 퍼짐
